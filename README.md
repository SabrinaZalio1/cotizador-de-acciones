# Interfaz web para graficar la cotización de una acción en tiempo real

<br>
<br>

 Proyecto creado con **Create React App**. Se utilizaron librerias y herramientas como **React Router**, **Highcharts**, React **Datepicker** y **Typescript**.

<br>

Cuando se ingresa a la plataforma, se encontrara un listado de acciones con sus siguientes datos: simbolo, nombre, exchange, moneda y tipo:
<br>
<br>
<img width="1392" alt="Captura de pantalla 2024-03-30 a la(s) 12 35 47" src="https://github.com/SabrinaZalio1/cotizador-de-acciones/assets/81325676/eea57e4f-8194-419c-a178-f079c022c16e">
<br>
<br>
Se podra filtrar por nombre y por simbolo y buscar cualquier accion dentro de esas categorias.
<br>
<br>
Al clickear en un simbolo de la tabla, este redireccionara a la pantalla del detalle de la accion seleccionada. En esta pantalla, se podra elegir el intervalo de tiempo para graficar los valores high de la accion.
<br>
<br>
<img width="1395" alt="Captura de pantalla 2024-03-30 a la(s) 12 40 41" src="https://github.com/SabrinaZalio1/cotizador-de-acciones/assets/81325676/dc164306-7e1f-4b1d-b3f7-fa8d953ec21a">
<br>
<br>
Tambien se podra elegir un periodo de tiempo para realizar el grafico de los valores:
<br>
<br>
<img width="1381" alt="Captura de pantalla 2024-03-30 a la(s) 12 42 04" src="https://github.com/SabrinaZalio1/cotizador-de-acciones/assets/81325676/99245e5c-4d21-4812-96db-129c5744212f">
